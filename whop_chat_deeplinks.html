How to use this page

Open this page in a browser on the device and tap a link — it should open the Whop app. Every link is under whop.com/joined/…, the path the app's verified App Links filter claims; the parser strips the joined segment, so each one resolves to the same destination as its canonical deeplink (e.g. /joined/messages?chat=… ≡ /messages?chat=…).

If a tap opens the browser instead of the app (common on the develop build, which isn't auto-verified on whop.com):
Settings → Apps → Whop (develop) → Open by default → enable Open supported links and add whop.com. Then taps route into the app.

Fallback that always works (bypasses link verification) — push the URL as the deeplink extra, or use each row's copy adb button:

adb shell "am start -n com.whop.whopapp.develop/com.whop.whopapp.MainActivity \
  --es deeplink '<URL>'"

Note: the URL is single-quoted inside the device command on purpose — an unquoted & is eaten by the phone's shell and truncates the link before &message=… (so it opens the chat but won't scroll). The copy adb buttons already emit this safe form.
