Whop — Android Install Referrer POC

Build a Google Play install referrer URL, then verify the app reads the payload back after install. The referrer is a single URL-encoded string Google Play stores at click time and hands to the app on first launch via the Play Install Referrer Library. Use it to carry attribution (UTM) and a post-install deeplink target through the install.

App package: com.whop.whopapp  ·  develop build: com.whop.whopapp.develop
Mirrors getAndroidPlayStoreUrl() — UTM defaults whop_web / app_banner, plus optional Whop deeplink params.

3 · Test on a device build (Firebase / sideload)

⚠️ A Firebase or sideloaded build can't return a real campaign referrer.

Firebase App Distribution installs outside the Play Store, so InstallReferrerClient reports the organic default — utm_source=google-play&utm_medium=organic — exactly like a sideload. A real campaign referrer only comes from a Play internal-testing track install (section 4). To exercise the full path on a Firebase build, use the non-production debug_install_referrer adb hook below.

1 · Push/install the build via Firebase App Distribution (App Tester), then watch the log
adb logcat -b all -s InstallReferrer
2 · Cold-launch — logs the real (organic on Firebase) referrer on app open
3 · Inject a test referrer through the parse/log path

Single-quote the value so the device shell keeps the &. Expect: I InstallReferrer: Install referrer (debug inject): <value>. The hook is a no-op on the production flavor.

Implementation — current POC (log-only)

The current build captures the referrer and logs it on app open (tag InstallReferrer); the analytics event is intentionally not wired up yet. A non-production adb hook injects a referrer for testing off-Play.

Gradle

implementation("com.android.installreferrer:installreferrer:2.2")

Capture + log on app open (real path — Play returns the value, organic off-Play)

val client = InstallReferrerClient.newBuilder(context).build()
client.startConnection(object : InstallReferrerStateListener {
    override fun onInstallReferrerSetupFinished(responseCode: Int) {
        if (responseCode == InstallReferrerClient.InstallReferrerResponse.OK) {
            Timber.tag("InstallReferrer")
                .i("Install referrer on app open: %s", client.installReferrer.installReferrer)
        }
        client.endConnection()
    }
    override fun onInstallReferrerServiceDisconnected() {}
})

Debug-inject hook (non-production only) — read in MainActivity onCreate / onNewIntent

if (BuildConfig.FLAVOR != "production") {
    intent.getStringExtra("debug_install_referrer")?.let { referrer ->
        Timber.tag("InstallReferrer").i("Install referrer (debug inject): %s", referrer)
    }
}
