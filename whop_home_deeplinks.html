How to use this page

Open this page in a browser on the device and tap a link — it should open the Whop app on the home feed. whop.com/home-feed is now a verified App Links path (added to the manifest in this change), so tapping resolves into the app. Each /joined/home-feed… variant resolves to the same destination — the parser strips the joined segment.

If a tap opens the browser instead of the app (common on the develop build, which isn't auto-verified on whop.com):
Settings → Apps → Whop (develop) → Open by default → enable Open supported links and add whop.com. Then taps route into the app.

Fallback that always works (bypasses link verification) — push the URL as the deeplink extra, or use each row's copy adb button:

adb shell "am start -n com.whop.whopapp.develop/com.whop.whopapp.MainActivity \
  --es deeplink '<URL>'"
